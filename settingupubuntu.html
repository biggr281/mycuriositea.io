How I Set Up My Virtual Machines

When I first started setting up my virtual machines (VMs) for cybersecurity studies, I thought I had everything figured outâ€”until I broke something by tinkering too much. Only then did I learn about the power of snapshots! In this blog post, I'll walk you through the steps and resources I use to create my VMs the way I like them.

Step 1: Downloading Ubuntu

I prefer Ubuntu for its balance of usability and powerful tools. You can download the latest Ubuntu ISO file here.

Step 2: Setting Up VMware

Setting up VMware recently has been tricky without creating an account. Luckily, I found a direct link to the VMware Player installer here (thanks to the Reddit community here).

Step 3: Installing Ubuntu in VMware

Installing Ubuntu in VMware is straightforward. If you need a video guide, this tutorial covers it well: How to Install Ubuntu 24.04 LTS on VMware.

Step 4: Configuring Ubuntu in the Terminal

Once Ubuntu is installed, I start by updating the package list and upgrading the system. Here's how: 


        $ sudo apt-get update
        $ sudo apt-get upgrade

This ensures all packages are up to date.

Step 5: Installing Essential Tools

I also install some basic tools to enhance the setup. For example, I like using Gnome Tweaks for GUI management: 


        $ sudo apt-get install gnome-tweaks

I also install Brave for privacy-focused browsing: 


        $ sudo apt install curl
        $ sudo curl -fsSLo /usr/share/keyrings/brave-browser-archive-keyring.gpg https://brave-browser-apt-release.s3.brave.com/brave-browser-archive-keyring.gpg
        $ echo "deb [signed-by=/usr/share/keyrings/brave-browser-archive-keyring.gpg] https://brave-browser-apt-release.s3.brave.com/ stable main"|sudo tee /etc/apt/sources.list.d/brave-browser-release.list
        $ sudo apt update
        $ sudo apt install brave-browser

For a lightweight text editor, I install Gedit:


        $ sudo apt-get install gedit

Step 6: Setting Up Development Tools

Next, I set up Visual Studio Code for coding. You can download the .deb package here, then install it: 


        $ sudo dpkg -i code_1.93.1-1726079302_amd64.deb

I also install Git for version control:


        $ sudo apt install git

Step 7: Installing AppImages

I prefer to use Flameshot for screenshots and Obsidian for note-taking. Both of these tools come as AppImages. After downloading, make them executable by right-clicking the file, selecting "Properties," and checking "Allow executing as program."

To ensure they run, install the necessary support: 


        $ sudo apt install libfuse2

Final Thoughts

This setup is my basic starting point for working with virtual machines. There are always more tools to explore, but this configuration gives me a solid foundation for work and learning. Plus, I never forget to take snapshots anymore!
