My Approach to Notekeeping Using Obsidian

Notekeeping has evolved significantly since my time studying, and frankly, the concept of building a “second brain” is something I find overwhelming. After doing some research, I’ve found a system that works for me and my needs, which I’m sharing in this post.

Why I Chose Obsidian

I decided to use Obsidian as my notekeeping tool because it supports offline usage, is highly customizable through plugins, and has an innovative way of linking notes. You can download it here.

Using Flameshot for Screenshots

For screenshots, I use Flameshot, which complements my workflow with Obsidian. You can download Flameshot here.

Tagging for Dynamic Note Organization

Instead of feeling overwhelmed by traditional organizational methods like creating a strict table of contents, I’ve found Obsidian’s tagging feature incredibly helpful. The dynamic structure that tags create allows for organic connections between topics as they develop over time.

In Obsidian, you can use simple tags like #exampletag to categorize notes. Alternatively, using [[LinkToNewPage]] creates a new blank page linked to the subject. Over time, these linked pages can serve as an index, centralizing knowledge in an intuitive way.

Markdown Essentials in Obsidian

Obsidian uses basic Markdown, which makes it easy to format notes. Here are some of the key Markdown commands I use:

  • Header 1: # header
  • Header 2: ## header
  • Bulleted Lists: Just use a dash - to start the list
  • Numbered Lists: Type 1. and hit space to start
  • Code Blocks: Use `` for inline code, or ```language for multiline blocks
  • Images/Embeds: Embed images using ![[filename]]

Folder Structure for Organizing Notes

When I started using Obsidian, I created a few essential folders to help me stay organized:

  • Notes
  • Logs
  • Excalidraw (for diagrams)
  • Calendar
  • Templates
  • Assets
  • Export

These folders keep my files categorized and work seamlessly with my tagging system.

Final Thoughts

While I’m still growing my system and adapting to these changes, Obsidian offers a powerful yet flexible approach to notekeeping. I’ll revisit this post in the future to share updates as my system evolves. For now, this workflow has made keeping notes more efficient and less overwhelming.
